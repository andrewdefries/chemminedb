# QuickStart:
########################################################################

To start an example use:

  'sh joelibMoleculeExample.sh'

  For further targets try:

  sh build.sh -projecthelp
  or
  build.bat -projecthelp
  and then
  sh build.sh SMARTSMatchingExample
  or
  ant>ant SMARTSMatchingExample
