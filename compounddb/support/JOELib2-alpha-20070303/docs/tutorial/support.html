Chapter 11. Support

For further question there are archived mailing lists and tracking systems available:

  • Bug tracking system: Bugs should be send to the bug tracking system.

  • Support request tracking system: Support requests should be send to the support request tracking system.

  • Help mailing list and Help archive: This list can be used for general questions and problems.

  • Developer mailing list and Developer archive: Developers and people with algorithm, feature, and design request should mail to this list. We recommend to have a at first a look at the source code, so we can discuss things with guessing or concrete wishes what can be technically really improved.
